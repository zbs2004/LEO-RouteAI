# LEO-RouteAI
LEO RouteAI 是一个面向 LEO 卫星网络的混合跨域路由系统，结合强化学习的域间决策和经典的路径规划方法，目标是提升跨域路由的延迟、跳数和负载平衡性能。  LEO RouteAI is a hybrid LEO satellite cross-domain routing system that combines reinforcement learning for inter-domain decision-making with classical path planning, aiming to improve delay, hop count, and load balancing in cross-domain routing.
